# Security Policy

## Supported Versions

I'd recommend using the latest possible version. 
| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :white_check_mark:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Please report any security concerns to me at **msz@msz.eu** or by opening an issue on GitHub.

This project is completely open source and is provided **AS IS**. 
I'll do my best to address any reported security concerns on a best effort basis.

## Additional Notes

In the **Releases** section, you can find a compiled executable for Windows.
Please **ALWAYS** check, after downloading and even before running it, that it's digitally signed by a public CA.
Currently, the executable is signed by: *Open Source Developer Marco Simone Zuppone*
